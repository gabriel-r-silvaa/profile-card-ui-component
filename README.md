# Profile Card UI Component

## 📌 Objective
This project is a simple profile card UI component built using HTML and CSS.  
The goal is to practice layout structuring, styling, and basic UI design principles.

---

## 🚀 Features
- Profile image with styled border
- User name and job title display
- Social media icon buttons
- Action buttons (Subscribe, Message)
- Basic analytics section (likes, comments, shares)
- Clean and modern card design

---

## 🛠 Technologies Used
- HTML5
- CSS3 (Flexbox)
- Boxicons (for icons)
- Google Fonts (Poppins)

---

## 📚 What I Learned
- Structuring semantic HTML for UI components
- Using Flexbox for layout alignment
- Creating reusable UI blocks
- Styling components with border-radius, shadows, and pseudo-elements
- Importing external fonts and icon libraries

---

## ⚠️ Limitations
- No responsiveness for different screen sizes
- No interactivity (buttons are not functional)
- Static content (no dynamic data)
- No accessibility improvements (ARIA, alt descriptions, etc.)

---

## 🔧 Future Improvements
- Add responsiveness using media queries
- Implement interactivity with JavaScript
- Improve accessibility (ARIA labels, alt text)
- Convert into a reusable component (React or Web Component)
- Add hover animations and transitions
